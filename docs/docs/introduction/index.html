The Project

This is a code challenge proposed by Kenat Software to evaluate programming techniques.

Challenge

The challenge of the code is to use an API available through the URL: https://swapi.co/

It should be known to all SW star ships to cover a certain distance how many refueling stops are required.

The application will have as input a distance in mega lights (MGLT). The output should be a collection of all starships and the total amount of stops needed to make the distance between the planets.

Technologies Used

To accomplish the challenge, NET Core 3.0 was used.

For the tests it was used unit test with MSTest and .NET Core.
